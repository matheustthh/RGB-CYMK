Modelos RGB e CMYK

RGB: O Modelo de Cor Aditivo

RGB (Vermelho, Verde, Azul) é um modelo de cor aditivo usado em displays digitais, como monitores de computador, smartphones e TVs. Neste modelo, as cores são criadas combinando diferentes intensidades de luz vermelha, verde e azul.

Pontos-chave sobre RGB:

  • Os pixels combinam luz vermelha, verde e azul para criar várias cores.
  • A intensidade de cada cor varia de 0 (desligado) a 255 (intensidade total).
  • Quando as três cores estão na intensidade máxima (255, 255, 255), vemos branco.
  • Quando as três cores estão desligadas (0, 0, 0), vemos preto.

O funcionamento dos pixels

Cada pixel em uma tela digital é composto por três subpixels: um vermelho, um verde e outro azul. A intensidade de cada subpixel determina a cor exibida:

  • Os subpixels podem ter vários níveis de brilho (0-255).
  • A combinação dessas intensidades de subpixels cria a cor final
  • Um exemplo é o vermelho (255) e verde (255) sem azul (0) cria amarelo

CMYK: O Modelo de Cor Subtrativo

CMYK (Cyan, Magenta, Yellow, Black) é um modelo de cor subtrativo usado na impressão. Ao contrário do RGB, que adiciona luz para criar cores, o CMYK funciona subtraindo luz através do uso de pigmentos de tinta.

Pontos-chave sobre CMYK:

  • Cyan, Magenta e Yellow são as cores primárias na impressão.
  • O "K" (Preto) representa a tinta preta, que é adicionada para melhor contraste e para economizar tinta colorida.
  • As cores são criadas misturando diferentes porcentagens dessas quatro tintas.
  • O branco é a ausência de tinta (a cor do papel), enquanto o preto é uma combinação de todas as tintas.

Comparando RGB e CMYK

Embora RGB e CMYK sejam modelos de cor, eles servem a propósitos diferentes e funcionam de maneiras fundamentalmente distintas:

RGB CMYK
Usado em displays digitais Usado na impressão
Adiciona luz para criar cores Subtrai luz usando pigmentos de tinta
Preto é ausência de luz Preto é presença de todas as tintas
Branco é todas as cores na intensidade máxima Branco é ausência de tinta (cor do papel)
